# Cooperative Housing Culture & Conflict Resolution Training

This repository contains comprehensive training materials for cooperative housing trustees, focusing on culture development and conflict resolution. The content is specifically tailored for college-educated audiences, emphasizing strategic analysis, practical application, and sophisticated problem-solving.

## Quick Start

1. **Interactive Web Tools**: Explore the web interface in `/web/` for interactive conflict resolution frameworks and policy development tools
2. **Training Materials**: Review the comprehensive training manual and workbook materials
3. **Activities & Exercises**: Use the facilitator guides and role-playing scenarios for hands-on learning

## Repository Structure

- **`/web/`**: Interactive web-based components and presentation materials
- **`/docs/`**: All training documentation and materials
  - **`/docs/workbook/`**: Participant workbook materials, worksheets, and case studies  
  - **`/docs/activities/`**: Facilitator guides for interactive activities and role-playing scenarios
  - **`/docs/`**: Main training manual and comprehensive guides
- **`/resources/`**: Policy templates and additional reference materials

## Key Features

- **Strategic Conflict Resolution Framework**: Interactive tools for navigating complex disputes
- **Communication Assessment Tools**: Self-evaluation instruments for governance effectiveness
- **Policy Development Assistant**: Customizable templates for cooperative policies
- **Role-Playing Scenarios**: Realistic simulations of common cooperative challenges
- **Case Studies**: In-depth analysis of actual cooperative housing situations

## Getting Started

1. Start with the [comprehensive training manual](cooperative-housing-culture-training/docs/cooperative-housing-trustee-training-manual.md) for foundational concepts
2. Explore the [interactive web tools](cooperative-housing-culture-training/web/) for hands-on learning
3. Use the [workbook materials](cooperative-housing-culture-training/docs/workbook/) for individual and group exercises
4. Implement the [policy templates](cooperative-housing-culture-training/resources/) in your cooperative
